# FileRunner Frontend

Next.js frontend for FileRunner. Coming in Phase 2.

## Planned Features

- User authentication (login/register)
- Dashboard with project management
- File browser and upload interface
- Admin panel
- Beautiful UI with shadcn/ui
- Dark mode support

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- TanStack Query
- Zustand
